## Setup

### Usage with npm

```bash
cd frontend
npm install
npm dev
```

### Usage with pnpm (Recommanded)

```bash
cd frontend
pnpm install
pnpm dev
```

Interface available at localhsot:8000

## Initialization backend

```bash
cd backend/app
python3 main.py
```
